## Getting started
1. Clone the repo:
   ```bash
   git clone https://github.com/oomfie-bot/CODESTACLE.git
   cd CODESTACLE
   ```
2. Make changes and commit:
   ```bash
   git checkout -b my-change
   # edit files
   git add .
   git commit -m "Describe your change"
   ```
3. Push your branch:
   ```bash
   git push -u origin my-change
   ```
